# aoc-arduino-stars

Display Advent of Code stars via an arduino and some led strips.

## Dev environment

- Arduino
  - Board: ARDUINO-UNO-R4-WIFI
  - Led strip chip: WS2813B
  - Download Arduino IDE
  - Install libraries within Arduino IDE:
    - ArduinoHttpClient (0.6.1)
    - ArduinoJson (7.2.0)
    - FastLED (3.7.8)
    - NTPClient (3.2.1)
    - Time (1.6.1)
